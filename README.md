# Port Scanner

⚡ A fast and lightweight port scanner for network reconnaissance and security testing.

## Features

- Fast TCP port scanning
- Detect open ports
- Simple command-line interface
- Lightweight and easy to use
- Suitable for learning network security concepts

## Installation

```bash

```

## Usage

```bash
python scanner.py <target-ip>
```

Example:

```bash
python scanner.py 192.168.1.1
```

## Requirements

- Python 3.x
