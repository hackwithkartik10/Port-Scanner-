# Port Scanner

⚡ A fast and lightweight port scanner for network reconnaissance and security testing.

## Features

- Fast TCP port scanning
- Detect open ports
- Simple command-line interface
- Lightweight and easy to use
- Suitable for learning network security concepts

## Installation

```bash
git clone https://github.com/hackwithkartik10/Port-Scanner-.git
cd Port-Scanner-
```

## Usage

```bash
python3 port.py
```

Example:

```bash
Enter Your IP/Domain = Examp 192.68.1.1
```

## Requirements

- Python 3.x
